Personal Finance Dashboard using Excel
## 🔹 Project Overview
This project focuses on analyzing personal financial data including income, expenses, savings, and investments using Excel and Power BI.

## 🔹 Tools Used
- Microsoft Excel (Pivot Tables, Charts)

## 🔹 Key Features
- KPI metrics: Savings Rate, Expense Ratio, Investment Ratio
- Monthly trend analysis
- Expense category breakdown
- Interactive dashboard with slicers

## 🔹 Insights
- Identified high spending in shopping category
- Analyzed savings fluctuations across months
- Maintained consistent investment strategy
